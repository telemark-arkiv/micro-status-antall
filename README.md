# micro-status-antall
Shows count for different systems

## API

### **/json**

Returns json of formatted data.

#### GET

```bash
$ curl https://antall.status.t-fk.win/json
```

### **/html**

Renders formatted data to html. 

#### GET

```bash
$ curl https://antall.status.t-fk.win/html
```
